checkData()

checkData(string $type, string $value, boolean $interval, int $uid) : int

Validates if the supplied user data token is valid

Parameters

string $type

Data type which much match a set data type in the database schema
string $value

The value to be checked
boolean $interval

Should the given token be checked against its life span
int $uid

The corresponding user to check against. If 0 then the current logged in users id is used, if greater than 0 it is taken as the users id. If NULL is used then the token is checked against all users

Returns

int —

If correct the users id is returned else false
